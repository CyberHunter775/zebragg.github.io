Hexo搭建个人博客

By zebragg

发表于 2015-10-02

动机

学习Android也有一段时间了，由于刚开始没有记录的习惯，一些在学习过程中遇到的很有意思的问题往往当时下了大力气去解决，但过一段时间再回头看代码，总是感觉生疏了很多，还得从新开始。这就产生了一个很严重的问题，学习进度太慢了。

这也是打算开始写博客的最主要的原因，把自己平时遇到的一些问题记录下来，毕竟知识重在积累！

难吗?

我是折腾了整整一天才把这个流程大概弄明白的，那么有同学就可能就要问了，是不是很复杂啊？我想说，如果找对了方法，其实搭建一个免费的个人博客是很简单的，真的很简单。不信你往下看….

准备干活之后，你肯定要去百度找找’ 如何搭建免费的个人博客 ‘，之类的关键字，不过可惜的是，搜到的很多质量都不是很高，看了之后，也仅仅是大脑中多了几个新概念，对于解决你如何搭建博客的问题，事实上并没有什么卵用，折腾了一上午然并卵之后，我选择了放弃，用我大Google了一把，瞬间找到了几篇高质量的博文，然后就可以开始写自己的博文了,haha..

推荐

  • 使用Hexo搭建个人博客(基于hexo3.0)
  • Hexo搭建Github静态博客
  • 博客主题-Jacman的使用

总结

具体怎么搭，好多博文里面写的要比我好多了，就不画蛇添足了，免得误导其他人，自己在这里就做一个大概的总结流程就足够了。

  • 准备
    • node.js
    • Git
    • GitHub账号
  • 流程
    • 安装 node.js
    • 安装 Git
    • 使用 Git 在指定的路径下安装 Hexo
      • npm install -g hexo (可能需要翻墙)
      • hexo init
      • hexo install
      • hexo generate/g
      • hexo server/s
    • 本地预览 localhost:4000
    • 将本地博客部署到 Github
      • 设置 SSH
    • hexo主题配置
      • 使用 Git下载主题
      • 进行配置
      • hexo deploy/d 发布博客到 Github
    • 主要注意问题
      • hexo 和 主题配置时 缩进一格
      • 生成 SSH 时没有必要使用密码，
