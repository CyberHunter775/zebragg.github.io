CSS使用

  • 内链样式表
    • <body style= "background-color:green"></body>
  • 嵌入式样式表
    • <style type = "text/css">xxxxx</style>
    • 需要写在 <head></head>里面
  • 引入式样式表
    • <link rel="StyleSheet" type="text/css" href="引入的css样式路径"></link>

定义样式表

  • HTML标记定义
  • Class 定义
    • <p class="xxx"></p>
    • .xxx 开始

  • ID 定义

    • <p id="xxx"></p>
    • ID定义是以 “#” 开始的，#xxx

  • 优先级：ID>Class>HTML,同级时选择里元素最近的一个

  • 组合选择器（同时控制多个元素） h1,h2,h3

CSS常见的属性

  • 颜色属性
    • color 定义文本的颜色，有多种表达形式，green | #ffffff | #fff | rgb()
  • 字体属性
    • font-size 字体大小
      • px 固定值
      • % 父元素的百分比
      • smaller 比父元素更小
      • larger 比父元素更大
      • inherit 继承父元素
    • font-family 字体
    • font-weight 字体权重
      • normal、bold、bolder、lighter
      • 100-900 之间的整数值
  • 背景属性
    • 背景颜色：background-color
    • 背景图片：background-image
      • background-image:url
      • background-image:none
    • 背景重复：background-repeat
    • 背景位置：background-position
  • 文本属性
    • 排列：text-align
      • left、center、right
    • 文本行高：line-height
      • % 基于字体大小的百分比行高
      • 数值 设定固定值
    • 首行缩进：text-indent
    • 字符间距：letter-spacing
    • 文本方向：direction
    • 文本大小写：text-transform

  • 边框属性

    • 边框风格：border-style
      • 单独定义某一方向边框样式
        • border-bottom-style 下边框样式
        • border-top-style 上边框样式
        • border-left-style 左边框样式
        • border-right-style 右边框样式
      • 边框风格样式属性值
        • none
        • solid
        • dashed
        • dotted
        • double
        • groove
        • ridge
        • inset
        • outset
        • inherit
    • 边框宽度：border-width
      • 单独风格
        • border-top|bottom|left|right
      • 属性值
        • thin | medium | thick | px | inherit
    • 边框颜色：border-color
      • 单独风格
        • border-top|bottom|left|right-color
          　属性值 　颜色值名称，RGB,RGBA,十六进制
    • border-color:
      • 一个值：border-color：（上、下、左、右）
      • 二个值：border-color：（上下） （左右）
      • 三个值：border-color:（上） （左、右） （下）
      • 四个值：border-color：（上）（右）（下）（左）

  • 列表属性

    • 标记类型：list-style-type

    • 标记的位置：list-style-position

      • inside 列表项目标记在文本内，
      • outside 默认值，标记位于文本的左侧，
      • inherit 规定从父元素继承
    • 设置图像列表标记：list-style-image
      • URL 图像的路径
      • none 默认
      • inherit 从父元素继承

DIV+CSS 布局

  • div 与 span 的存在为了应用 CSS 样式
  • div 和 span 的区别：span 是内联元素，div 为块级元素

  • 盒子模型

    • 内容-内边距-边框-外边距

  • 相关属性

    • z-index 层覆盖的先后顺序
    • display 显示属性
      • display:none 层不显示
      • display:block 块级显示
      • display:inline 内联显示
    • float 浮动显示
      • left | right
    • clear 清除浮动
    • overflow 溢出处理
      • hidden 隐藏超出层的大小
      • scroll 无论内容是否超出层的大小都添加滚动条
      • auto 超出时自动加载滚动条
