HTML 的基本结构

  • <html></html> HTML的文档标记
  • <head></head> HTML的头标记，用来包含文件的基本信息，标记内容不会在浏览器中显示
  • <title></title> HTML的文件标题
  • <body></body> HTML的主体标记
  • <meta></meta> 页面的元信息，提供页面的元信息，在 <head>标签内

HTML常用标签

  • <br> 换行，HTML中输入的 空格和 回车是没有作用的
  • <p> 换段落
  • <center> 居中对齐
  • <pre> 预格式标记

  • 列表

    • <li> 列表项目
    • <ul> 无序列表

    • <ol> 有序列表，可以显示特定的一些顺序 <ol type="1|a|A|I|i|类型" ></ol>

    • <dl><dt><dd> 定义型列表，对列表条目进行简短说明
      `

      <dt>第一章</dt>
<dd>第一节</dd>

      `

  • <hr> 水平线
  • <div> 层标记，层与层之间可以嵌套
  • <h1>..<h6> 标题标记，依次对应不同的标题大小
  • 字体标记
    • <b> 字体加粗
    • <i> 斜字体
    • <sub> 文字下标
    • <sup> 文字上标
    • <tt> 打印字体
    • <cite> 引用
    • <em> 字体强调
    • <strong> 表示强调
    • <small> 小型字体
    • <big> 大型字体
    • <u> 下划线>
  • 图像标记 <img>
    • 为单标签
    • 格式：<img src="img path" width="宽度" height= "高度" border= "边框">

超链接的使用

  • <a href= "/" target= "打开方式" name= ""></a>
  • href 链接的地址，可以为一个网页，也可以是一个视频、图片、音乐等
  • target 链接打开的方式
    • _blank 在一个新窗口中打开
    • _seif 默认，在当前窗口中打开
    • _parent 在父窗口中打开
    • _top 在顶层窗口中打开
  • name 指定页面的锚点名称

表格

  • <table 属性1 属性2>表格内容</table>
    • width 表格宽度 可以使用 % 表示占父元素的百分比
    • height 表格高度
    • align 表示表格的位置
    • border 表示表格的外边框
    • cellspacing 单元格之间的间距
    • cellpadding 单元格内容和单元格边框的间距
    • frame: 控制表格边框的四条边框
      • void 无边框，默认
      • above 表示仅顶部
      • below 表示仅底部
      • hside 顶部+底部
      • lhs 仅左部
      • rhs 仅右部
      • vside 左+右
      • box 全部
      • boder 全部
    • rules 控制是否显示以及如何显示单元格之间的分割线
      • none 表示无分割线（默认）
      • all 全部分割线
      • rows 仅有行分割线
      • clos 仅有列分割线
      • groups 仅在行组和列组之间有分割线
  • <caption> 表格标题，位置在 table 标签之后，tr 标签之前
    • align top | bottom | left | right 分别为表格标题在表格的位置，上下左右
  • tr 定义表格的一行，<tr></tr> 行内可以嵌套 多个 或
    • bgcolor 背景
    • align 对齐方式（垂直方向）
    • valign 设置水平方式的对齐
  • td 与 th 必须嵌套在 <tr>内
  • 区别
    • th 为表头标记，位于首行或是首列，文字默认会被加粗，td不会
    • td 是数据标签，为该单元内的具体数据
  • 共同
    • 属性标记是一样的
  • 属性
    • bgcolor 背景
    • align 对齐方式
    • valign 水平对齐
    • width 宽度
    • height 高度
    • rowspan 单元格所占的行数
    • colspan 单元格所占的列数

表单设计

  • <from></from> 表单提交时的内容就是 <from>表单中的内容
  • <from action= "介绍表单内容的地址" name= "表单名称" method= "post|get"></from>
  • 常用属性
    • name 表单名字
    • method 提交方式，get post
      • get 表单内容会附加在 URL 之后，会限制提交内容的长度，
      • post 将表单中的数据一并包含在表单主体中，一起传送到服务器，数据没有大小限制
    • action 表单数据处理的 URL ，如果为空则为当期的文档的URL地址
    • enctype 设置编码方式
    • target 用来指定目标窗口
  • <input> 标记
    • 没有结束标记
    • <input type= "类型" name= "" value= "" ..>
    • type= "text" 表示一个文本输入域
    • type= "password" 表示一个密码输入
    • name 定义控件的名称
    • value 初始化值，
    • size 控件的长度
    • maxlength 输入框中允许的最大长度
    • 按钮
      • type= "submit" 为提交按钮
      • type= "reset" 为重置按钮
      • type= "button" 为普通的按钮
    • 单选框与复选框
      • type= "radio" 为单选按钮
      • type= "checkbox" 为复选按钮
      • 可以使用 check 属性来设置默认的选中项
  • <textarea> 多行文本域
    • <textarea name,rows,cols value>
    • rows,cols 舒心分别用来指定显示的行数与列数，单位为字符个数

  • <select> 菜单下拉域

    `<select name="address">
    <option value="四川">四川</option>
    <option value="四川">四川</option>F
    <option value="四川">四川</option>
</select>`
